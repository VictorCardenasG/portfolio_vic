Emotion Classification Using Deep Learning

Role

Data Scientist

Year

2025

Client

Western Institute of Higher Education

Introduction

The objective of this project is to develop a deep learning model that can classify human emotions from images. The model is trained on a dataset of facial images categorized into six emotion classes: Anger, Fear, Happy, Neutral, Sad, and Surprise. The project utilizes a convolutional neural network (CNN) architecture to achieve this task, leveraging the ResNet-152 backbone for feature extraction.

The Challenge

Present datasets on emotion classification fail to detect emotions using simple deep learning architectures.

The Solution

Manually develop a highly diversified dataset which allows to train using a simply ResNet-18 architecture achieving high accuracy and F1 scores.
