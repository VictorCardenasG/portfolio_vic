Predicting Spotify's Most Streamed Songs Using Audio Features

Role

Data Scientist

Year

2024

Tools

Python, Spotify API, Scikit-learn

Introduction

In this project, I set out to predict whether a song would be among the most streamed songs based on its various audio features. Utilizing data analytics and machine learning techniques, I developed a model capable of identifying songs with similar characteristics to the most streamed ones on Spotify.

The Challenge

Understanding what makes a song popular on Spotify among millions of tracks, and identifying the key audio features that contribute to high streaming numbers.

The Solution

A machine learning model that analyzes audio features to predict streaming potential, using Isolation Forest to identify songs with characteristics similar to current hits.

Process

The project involved several key phases:

01

Data Collection

Gathered Spotify's most streamed songs and their audio features through the Spotify API.

02

Data Preparation

Cleaned, normalized, and merged datasets for analysis.

03

Exploratory Analysis

Identified patterns and correlations between audio features and streaming numbers.

04

Model Development

Built and trained an Isolation Forest model to identify popular song characteristics.
